미디어쿼리

브라우저의 화면 너비에 반응하여 css를 다르게 적용할 수 있다.

각 기계(디바이스) 화면너비의 지점을 break point라고 부른다.

일반적으로 사용되는 브레이크포인트

  • 모바일 ( ~ 최대 768px)
  • 태블릿 (최소 768px ~ 최대 1024px)
  • 데스크탑 (최소 1024px ~ )

위의 일반적인 break point는 반드시 지켜야 하는 것이 아니다. 왜냐하면 기계는 계속 새로 생겨나고 있기 때문이다.

우리가 pc화면의 디자인을 할 때 만약 컨텐츠가 들어가는 공간 컨테이너 너비를 1400px로 했다면 break point를 1400px을 기점으로 다른 레이아웃이 적용될 수 있도록 유동적으로 break point를 설정할 수 있다.

실제 예로 큰 회사에서는 키디어쿼리를 30가지로 나누어서 관리하는 곳도 있다.

미디어쿼리 작성 시 띄어쓰기 주의 / (괄호)앞뒤로 띄어쓰기 할 것

괄호 안의 구간은 겹치면 안된다.
