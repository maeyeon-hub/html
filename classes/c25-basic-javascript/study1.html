자바스크립트란?

홈페이지에 동적인 컨텐츠를 생성하거나 ui의 기능을 만들 때 사용하는 언어

자바스크립트 언어를 사용하기 위해서는 script 태그를 만들어서 명령을 할 수 있다. 이때, script 태그의 위치는 어디든 관계없다.

script태그의 시작태그와 마침태그 사이의 공간은 자바스크립트 문법만 사용할 수 있는 공간이다

자바스크립트 언어도 기본적으로는 위에서 아래 왼쪽에서 오른쪽 순서로 해석되고, 해석되는 순서에 맞춰서 명령이 실행된다.

각 명령을 적용한 후 마지막에 ;(세미콜론)을 사용하여 눈에 보이게 명령이 끝난다는 것을 표현해야한다.

그러나 줄바꿈을 한 경우는 암묵적으로 세미콜론을 사용하지 않아도 명령이 끝났다고 표현되기 때문에 생략이 가능하다.
