배경 - background

배경은 컨텐츠가 없거나 높이가 없으면 표현되지 않는다.

배경 스타일이 들어가는 공간은 태그의 가장 밑바닥이다.(컨텐츠박스보다 아래에 위치한다.)

background-color: blue; padding: 20px; height: 300px; width: 300px; box-sizing: border-box;

background-color

배경색상 입력

태그의 공간(테두리 안쪽 모든 공간)에 배경색을 나타낼 때 사용하는 css속성

margin: 100px 0; height: 80vh; background-color: bisque; background-image: url(../img/liza-azorina-1EwWP0we9sQ-unsplash.jpg); background-repeat: no-repeat; background-position: 100px 200px;

background-image

배경이미지 입력 - url()도구 사용

배경이미지는 입력하면 자동으로 가로, 세로로 반복된다.

background-repeat

배경이미지가 반복되는 속성

repeat : 기본값-가로, 세로 반복

no-repeat : 반복안함

repeat-x : 가로만 반복

repeat-y : 세로만 반복

background-position

배경이미지의 위치를 변경할 때 사용하는 속성

background-position : x좌표:y좌표

키워드 방식, %방식, 픽셀방식

height: 100vh; background-color: aquamarine; /* 필수로 넣어야하는 거 아님, 태그의 위치를 알기 위함 실전에서는 사용안해도 됨*/ background-image: url(../img/29c37f7770b02880045ca3d20f2bfe8a.jpg); background-repeat: no-repeat; background-size: 300px auto; /* 배경의 사이즈 조정 속성, 기본값은 auto로 설정되어 있음(즉 이미지의 원본사이즈)*/ /* background-size: auto 300px; */ /* background-size: 100% auto; */ /* background-size: auto 100%; */ /* background-size: cover; cover : 배경 전체를 커버하여 태그의 공간을 채우게 됨(이미지 잘림 있을 수 있음) */ background-size: contain; /* contain : 이미지가 태그 공간안에 잘리지 않는 이미지의 최대 크기로 채우는 방식 */ background-position: 50% 50%; /* ////////////////// 축약형 ////////////////// */ background: aquamarine url(../img/29c37f7770b02880045ca3d20f2bfe8a.jpg) no-repeat 0 0 /cover; /* background-size 앞에는 꼭 '/(슬래시)' 입력 [순서 지켜야 함] 색 이미지경로 반복 좌표 /사이즈 */ /* background는 축약현으로 쓸때 사용하는 속성 */

cover

cover방식은 배경이미지크기를 이미지가 잘려나가도 태그의 공간을 가득 매우는 방식

contain

contain방식은 배경이미지가 잘려나가지 않는 선에서 배경이미지의 크기를 최대로 키워서 넣는 방식

축약형 표현

background:배경색 이미지경로 반복 좌표 / 사이즈 순서대로 씀(순서 중요)
