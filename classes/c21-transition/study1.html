transition - 전환

중간수치로 표현 가능한 모든 css속성들에 대해서 처음과 끝 사이의 중간스타일을 자동으로 애니메이션처럼 만들어주는 속성

변환하기 전 스타일에 설정할 것

부드럽고 천천히 전환하는 스타일을 적용하는 것

개별 속성

  • transition-property : 전환속성을 지정할 css속성 선택(all모든css선택)
  • transition-duration : 전환스타일을 몇 초 동안 보여줄 것인지 설청(1s:1초, 0.5s:0.5초)
  • transition-timing-function : 전환 애니메이션의 속도감을 설정 (linear : 일정한 속도)/(ease,ease-in,ease-in-out)
  • transition-delay : 전환스타일을 몇 초 후에 실행할 것인지 설정(s, ms 단위 사용 가능)
