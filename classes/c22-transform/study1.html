변형 - transform

transform이라는 css속성은 보통 애니메이션 효과를 만들 때 사용한다.

기본적으로 너비나 높이의 기준이 명확한 박스형태의 태그에만 사용가능하다.

translateX : 가로축으로 이동하는 속성
단위는 픽셀, % 등 여러단위로 사용할 수 있고, %단위를 썼을 때 부모태그 기준이 아닌 선택한 태그의 너비에 비례하는 값만큼 이동
translateY : 세로축으로 이동하는 속성
부모태그 기준이 아닌 선택한 태그의 높이에 비례하는 값만큼 이동
translateZ : z축 입체축을 따라서 앞뒤로 이동시키는 속성
앞으로 이동은 +, 뒤로 이동은 - 뱡향으로 이동 시킬 수 있다. 속성을 적용하기 위해서 부모태그에 perspective속성을 사용하여 원근감을 만들어주어야 입체이동이 표현된다.
rotate : 평면회전, 각도단위 deg단위를 사용하여 회전시키는 속성
이때 변형의 기준도 변경하고 싶다면 transform-origin 속성을 사용하면 된다. 기본값은 50% 50%이다.
