boolean 데이터

값이 두개 밖에 없다. true/false

비교를 통하여 참거짓이 계산된다

크다,작다,같다,같지않다 와 같은 비교를 통하여 참거짓이 계산된다

!는 자바스크립트에서 부정의 의미 (not)으로 사용된다.

계산과 같은 형식이 아닌 [있다, 없다] / [진실, 거짓] 라는 형식에도 사용할 수 있다.

있으면 true 없으면 false

false로 간주되는 경우

  • 숫자데이터인데 값이 0인 경우
  • 문자데이터인데 내용이 없는 빈문자인 경우
  • 변수를 선언했는데 변수 안에 아무 값도 저장되지 않은 경우
  • * underfined 값은 false로 간주한다.
  • * null값은 false로 간주한다.
